HTML

HTML es un lenguaje de marcado utilizado en la construcción de páginas web. Los navegadores pueden interpretar los documentos HTML.

CSS

Las hojas de estilo en cascada son un mecanismo para agregar estilo a un documento web.

JS

JavaScript es un lenguaje de programación interpretado de secuencias de comandos de alto nivel estructurado, multiparadigma, débilmente tipado dinámicamente.

GIT

Git hub en inglés británico es un sistema de control de versiones distribuido, que se usa principalmente en el desarrollo de software, pero se puede usar para registrar el historial de edición de cualquier tipo de archivo.

React JS

React es una biblioteca JavaScript de código abierto enfocada en la creación de interfaces de usuario en páginas web.
